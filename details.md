# Commonds

### Install Commands
`composer global require laravel/installer`
`composer install`
`npm install vue`
`npm install vue-router`
`composer require inertiajs/inertia-laravel`
`npm install @inertiajs/vue3`

### PHP Commands
`php artisan serve`
`npm run dev`
`php artisan make:controller <controller-name>`
`php artisan route:list`
